# 内存动态壁纸

一个Windows桌面壁纸，将进程的内存内容映射成一个不断滚动的，五彩斑斓的网格，你看见的壁纸即这个程序本身

A Windows desktop dynamic wallpaper that maps the memory contents of processes into a continuously scrolling, colorful grid, and the wallpaper you see is the program itself
